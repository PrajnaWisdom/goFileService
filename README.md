# goFileService

**goFileService**是一个用**Go(Golang)**编写的文件上传下载的服务

## 技术栈

- [Gin](https://github.com/gin-gonic/gin)
- [Gorm](https://github.com/go-gorm/gorm)
- Redis
- Mysql
- Sqlite3

## 功能

- [x] 单文件上传
- [x] 大文件分片上传
- [ ] 取消上传
- [x] 单文件流式下载
- [ ] 大文件分片下载
- [x] 大文件MD5校验
- [x] 多账号上传
- [x] 上传访问控制
- [ ] 授权下载
